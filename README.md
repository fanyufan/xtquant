# xtquant

> 当前归档版本：**xtquant_250516**

xtquant — 迅投 MiniQMT Python 量化接口非官方镜像。提供行情数据（xtdata）与实盘交易（xttrader）双模块，支持历史/实时 K 线、分笔、财务数据订阅获取，以及报单、撤单、资产持仓查询等全链路交易 API。兼容 Python 3.6–3.13，需配合 MiniQMT 客户端运行。 ⚠️ 核心库版权归原厂商（迅投）所有，本仓库仅做学习使用。
